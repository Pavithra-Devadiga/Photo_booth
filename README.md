# 📸 PhotoBooth Web App

A fun and interactive **PhotoBooth web application** that allows users to capture photos using their webcam and generate a downloadable **photo strip**.

The app includes live camera preview, aesthetic filters, countdown before capture, shutter sound, and the ability to download the final photo strip.

🌐 **Live Demo:**  
https://pavithra-devadiga.github.io/Photo_booth/

---

## ✨ Features

📷 Webcam access directly from the browser  
⏱️ 3-second countdown before capturing a photo  
🎨 Multiple photo filters  
- Normal  
- Black & White  
- Sepia  
- Vintage  

📸 Capture **4 photos sequentially**  
🔊 Camera shutter sound effect  
🖼️ Automatically generates a **vertical photo strip**  
⬇️ Download the final photo strip as a **PNG image**

---

## 🛠️ Technologies Used

- **HTML** – Structure of the web page  
- **CSS** – Styling and responsive layout  
- **JavaScript** – Camera interaction and capture logic  

Browsers allow web applications to access the camera using the **MediaDevices API**, enabling real-time video capture and processing directly in the browser.
